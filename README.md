# go-barber-backend
Barber appointments management back-end made with NodeJS using Typescript

# Steps to run:
  - Run "yarn"
  - Run "yarn start"
  
## Routes
  ## Appointments
    - GET /appointments
    - POST /appointments
  ### Users
    - POST /users
    - Patch /users (to update Avatar)
  ### Sessions
    - Create Session (to Login)

Authentication with JWT token
